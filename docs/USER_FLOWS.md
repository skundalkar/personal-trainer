# User Flows

## Flow 1 — First session

1. The user chooses a primary goal, body area, time, equipment, experience level, and preferred coaching tone.
2. The app offers three exercises, each with a “why this” explanation and two alternatives.
3. The user chooses an exercise and can start it, save it, or replace it.
4. At the end, the user records completion, effort, enjoyment, and optional discomfort.
5. The next recommendation changes only for explainable reasons.

## Flow 2 — Form-coached exercise

1. The user selects a camera-supported exercise.
2. The app explains camera angle and asks the user to confirm a full-body view.
3. The app checks landmark confidence; it asks for better framing if uncertain.
4. The app gives one highest-priority cue at a time, plus positive reinforcement when appropriate.
5. The user ends the set and receives a short, non-diagnostic summary.

## Flow 3 — Returning after time away

1. The app welcomes the user without guilt: “Good to see you. Want a short lower-body session today?”
2. The user chooses a shorter, normal, or recovery-intensity option.
3. Recommendations favor previously enjoyed, lower-friction exercises unless the user selects a new goal.
4. The app asks whether to resume music and the prior coaching tone.

## Flow 4 — Music

1. The user selects a mood (calm, steady, energetic, or custom).
2. The user can select an existing playlist or connect Spotify.
3. The app requests playback only after an explicit user action.
4. If no service is connected, the app shows a saved link or starts the session silently.
